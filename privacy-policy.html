Privacy Policy – Sayaka Miki Image Finder

Last updated: 20 December 2025

Developer Information

This application is developed and operated by Sayaka Miki.

Information Usage

The application only accesses publicly available data from Pinterest, such as images, pin titles, descriptions, and metadata. This data is used strictly for image search and display purposes. No personal user data, including emails or passwords, is collected or stored.

Purpose

The data retrieved is used solely to:

  • Search and display images from Pinterest based on user keywords
  • Help users discover visual inspiration and relevant content
  • Redirect users to the original Pinterest content

Data Storage

The application does not permanently store Pinterest user data. Temporary caching may be used to improve performance.

Data Sharing

No data is shared with third parties.

Security

Reasonable technical measures are in place to protect access tokens and API usage.

Compliance

This application complies with Pinterest Developer Terms and API policies.
